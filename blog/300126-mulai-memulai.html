Karena gw bisa liat perubahan kode yang ditulis secara langsung tinggal buka file HTML di folder komputer, terus dicheck apakah ada error atau tidak. Soalnya kemaren di github gw kayak musti nunggu proses apa gitu namanya, dan ribet kalo buat bolak balik. Yes VS Code lebih oke sejauh ini.
